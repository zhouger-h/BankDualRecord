## 三、核心模块说明

### 3.1 DualRecordWidget（录制控件）

**状态机**

```
Idle ──[开始录制]──► Recording ──[暂停]──► Paused
  ▲                     │                    │
  │◄────[停止录制]───────┘◄───[继续]──────────┘
  │                     │
  └──[停止录制]──────────┘
```

**关键行为**

| 动作 | 摄像头 | 录制器 |
|------|--------|--------|
| 点「开始录制」 | openCamera() | recorder.record() |
| 点「暂停」 | 保持开启 | recorder.pause() |
| 点「继续」 | 保持开启 | recorder.record() |
| 点「停止录制」 | closeCamera() | recorder.stop() → 入上传队列 |

**文件列表**：`QListWidget` 每项内嵌 `QPushButton("▶ 播放")`，点击后调用 `playVideo(path)` 弹出 `PlayerWindow`，**无全局删除按钮**（文件由 UploadManager 上传成功后自动删除）。

---

### 3.2 PlayerWindow（独立播放器）

- 基于 `QMediaPlayer` + `QVideoWidget`
- 支持：播放/暂停、进度拖动、音量调节、全屏
- 键盘快捷键：空格=播放暂停，←/→=±5秒，F=全屏，Esc=退出全屏/关闭
- TCP PLAY 命令 → `DualRecordWidget::playVideo()` → `new PlayerWindow(path)` → 弹出
- `setAttribute(Qt::WA_DeleteOnClose)` — 关闭自动释放内存

---

### 3.3 TcpCommandServer（TCP服务）

- 监听 `0.0.0.0:9527`（可配置）
- 帧：`4字节大端长度 + JSON`
- 支持多客户端同时连接（通过 `ClientSession` 管理）
- 双向通信：请求-响应 + 事件主动推送

---

### 3.4 UploadManager（上传管理器）

**核心原则**：**上传成功才删除本地文件**；失败不删除，应用重启后自动续传。

**令牌桶算法**（带宽限速）

```
每秒补充令牌 = bandwidth_limit_kb × 1024 字节
每次写入前消耗等量令牌，桶空则阻塞等待
```

**断点续传**（队列持久化）

```
文件路径：~/BankDualRecord/.upload_queue.json
格式：[{"local_path":"...","remote_path":"...","retry_count":0,...}]
```

**指数退避重试**

```
延迟 = 30 × 2^min(retryCount, 4)  秒
重试次数达到 MAX_RETRY(5) 后移到队尾，下次启动时重置计数再试
```

---

### 3.5 InitConfigDialog（首次配置向导）

- 在 `main.cpp` 中判断：`~/.config/BankDualRecord/config.ini` 不存在时弹出
- 分两个 Tab：**缓存&服务** / **上传服务器**
- 保存后写入 ini 文件，不再弹出

## 五、配置文件格式

路径：`~/BankDualRecord/config.ini`

```ini
[cache]
video_path = D:/DualRecord/Cache

[server]
tcp_port = 9527

[upload]
protocol = sftp              ; ftp 或 sftp
ftp_host = 192.168.1.100
ftp_port = 22
ftp_user = uploaduser
ftp_pass = P@ssw0rd
ftp_remote_dir = /dualrecord/
bandwidth_limit_kb = 1024   ; KB/s，0=不限速
```

## 七、开机自启实现

### Windows

```nsi
; NSIS安装脚本写入注册表
WriteRegStr HKCU "SOFTWARE\Microsoft\Windows\CurrentVersion\Run" \
    "银行双录控件" '"C:\Program Files (x86)\BankDualRecord\DualRecordService.exe"'
```

### 麒麟/信创

双重保障：
1. `systemd` 系统服务（`/etc/systemd/system/dualrecord.service`）
2. XDG autostart（`~/.config/autostart/BankDualRecord.desktop`）
